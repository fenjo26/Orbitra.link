# Orbitra v0.9.1 Tracker

Добро пожаловать в документацию проекта Orbitra! 

Orbitra — это упрощённый клон популярного трекера Keitaro. Система предназначена для управления и отслеживания трафика с возможностями сплит-тестирования, ротации потоков и точного учёта конверсий.

## Навигация по документации

Мы разделили информацию на несколько логических разделов для удобства чтения:

1. **[Архитектура и Технологии](architecture.md)** — описание стека (PHP + React + SQLite) и структуры базы данных.
2. **[Основные Функции](features.md)** — как работают кампании, дашборд, интеграции и Telegram-бот.
3. **[REST API](api.md)** — документация по всем доступным эндпоинтам для интеграции.
4. **[Запуск и Деплой](deployment.md)** — инструкции по разработке, запуску через Docker и сборке для продакшена.

## Краткий обзор возможностей

- **Без лишних зависимостей:** База данных SQLite создается на лету, не нужен MySQL.
- **Современный UI:** Написан на React + Vite с плавной навигацией и чистым дизайном.
- **Гибкость:** Поддержка 6 моделей оплаты (CPC, CPM, CPA и др.) и более 30 передаваемых параметров.
- **Telegram Bot:** Встроенный бот для получения мгновенных уведомлений об успешных конверсиях.
