# Развертывание и Запуск Orbitra

Проект Orbitra оптимизирован для быстрого старта без сложной настройки окружения. В основе лежит встроенный сервер PHP и готовая конфигурация Docker. 

## Быстрая Автоматическая Установка (Production)

Самый быстрый и правильный способ запуска проекта на сервере — использование встроенного инсталлятора. 

1. Убедитесь, что у вас "чистый" Linux сервер (Ubuntu 20.04/22.04).
2. Выполните команду от имени root/sudo:
   ```bash
   wget -qO- https://raw.githubusercontent.com/fenjo26/Orbitra.link/main/install.sh | bash
   ```
3. Скрипт сам установит Nginx, PHP 8+, SQLite, скачает код фронтенда и настроит сертификаты HTTPS.

> **Стандартные доступы (создаются инсталлятором):** `Логин: admin`, `Пароль: admin`

## Локальная Разработка (React & PHP Server)

Если вы планируете изменять исходный код фронтенда (`React`) и визуальную часть трекера, вам потребуется локальная связка PHP + Vite.

### Часть 1: Запуск Backend API
Откройте первый терминал и запустите встроенный PHP сервер с роутером:

```bash
cd Orbitra
php -S localhost:8080 router.php
```
*Этот сервер будет принимать API запросы и редиректы.*

### Часть 2: Запуск Frontend (Vite)
Откройте второй терминал, установите npm пакеты и запустите dev-сервер:

```bash
cd Orbitra/frontend
npm install
npm run dev
```

После этого откройте в браузере `http://localhost:5173`. Все API запросы с фронтенда будут автоматически проксироваться (через `vite.config.js` proxy) на `http://localhost:8080/api.php`, что позволяет избежать проблем с CORS и сессиями в режиме разработки.

## Сборка для Продакшена (Production Build)

После того как все изменения во фронтенде завершены, необходимо "собрать" React приложение в статические JS/CSS файлы, которые будет отдавать ваш Nginx/Apache или встроенный роутер.

```bash
cd Orbitra/frontend
npm run build
```

Эта команда сгенерирует оптимизированные файлы в папку `frontend/dist/`. Загрузите весь проект Orbitra (включая папку `frontend/dist`) на ваш VPS сервер.

## Настройка Интеграции Telegram Bot

1. Откройте Telegram и найдите официального бота `@BotFather`.
2. Введите команду `/newbot`, следуйте инструкциям и получите токен (API Token) вида `1111:AAABBBCCCDDD`.
3. Откройте панель администратора Orbitra -> **Интеграции -> Telegram Bot**.
4. Вставьте токен в поле и нажмите **Подключить**. Ваш домен (откуда вы сидите) автоматически установится как **Webhook** для Телеграма.
5. Напишите созданному боту `/start` из вашего Телеграмм аккаунта. 
6. Включите переключатели уведомлений на той же странице интеграций.
