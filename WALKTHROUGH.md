## 1. Установка и запуск

### Вариант А: Локальная разработка

```bash
# Клонируйте проект
cd /path/to/your/projects

# Терминал 1: Запуск backend
cd LTT
php -S localhost:8080 router.php

# Терминал 2: Запуск frontend (для разработки)
cd LTT/frontend
npm install
npm run dev
```

### Вариант Б: Docker

```bash
cd LTT
docker-compose up -d
```

### Вариант В: Продакшн

```bash
cd LTT/frontend
npm install
npm run build
# Затем настройте веб-сервер на папку LTT
```

## 2. Первый вход

1. Откройте браузер: `http://localhost:8080`
2. При первом запуске система создаст администратора автоматически
3. Введите данные:
   - **Логин**: `admin`
   - **Пароль**: `admin`
4. Нажмите "Войти"

> ⚠️ **Рекомендация**: Смените пароль после первого входа в разделе "Администрирование → Пользователи"

---

## 3. Создание домена

Домены используются для трекинговых ссылок.

1. В меню выберите **Домены**
2. Нажмите **+ Добавить домен**
3. Заполните поля:
   - **Имя домена**: `tracker.example.com` (без https://)
   - **Индексная кампания**: (опционально) кампания для главной страницы
   - **Catch 404**: (опционально) редирект 404 на кампанию
4. Нажмите **Сохранить**

---

## 4. Создание партнерской сети

Партнерская сеть — это место, откуда приходят выплаты за конверсии.

1. В меню выберите **Партнёрские сети**
2. Нажмите **+ Добавить сеть**
3. Выберите шаблон (например, **Leadbit**) или заполните вручную:
   - **Название**: `My Affiliate Network`
   - **Postback URL**: URL для отправки постбеков в сеть
   - **Параметры оффера**: `&subid={subid}&ip={ip}`
4. Нажмите **Сохранить**

### Доступные шаблоны сетей:
- Leadbit
- M4Leads
- Dr.Cash
- AdCombo
- Partners1xBet
- Traffic Light
- LemonAD
- MelbetAffiliate

## 5. Создание оффера

Оффер — это конечный URL, куда направляется трафик.

1. В меню выберите **Офферы**
2. Нажмите **+ Создать оффер**
3. Заполните основные поля:
   - **Название**: `My Offer`
   - **URL**: `https://offer.example.com/?subid={subid}`
   - **Партнёрская сеть**: выберите созданную сеть
   - **Группа**: (опционально)
4. Настройте выплаты:
   - **Тип выплат**: CPA (за действие) или CPC (за клик)
   - **Значение выплаты**: `10.00`
   - **Автовыплата**: включить для получения данных из постбека
5. Настройте каппинг (ограничение):
   - **Лимит конверсий**: `100` (в день)
   - **Часовой пояс**: `Europe/Moscow`
6. Нажмите **Сохранить**

### Макросы для URL оффера:
| Макрос | Описание |
|--------|----------|
| `{subid}` | ID клика |
| `{ip}` | IP адрес |
| `{country}` | Страна |
| `{device}` | Устройство |
| `{keyword}` | Ключевик |

## 6. Создание кампании

Кампания — это основная единица трекинга.

1. В меню выберите **Кампании**
2. Нажмите **+ Создать кампанию**
3. Откроется редактор кампаний

### Вкладка "Основные"

```
┌──────────────────────────────────────┐
│ Название кампании                    │
│ [My Campaign                   ]     │
│                                      │
│ Алиас *                              │
│ [j4bCV6mX] [🎲]                      │
│                                      │
│ URL кампании                         │
│ [https://tracker.example.com/j4bCV6mX] [📋] │
│                                      │
│ Домен                                │
│ [▼ tracker.example.com         ]     │
│                                      │
│ Группа                               │
│ [▼ Без группы                   ]    │
│                                      │
│ Источник трафика                     │
│ [▼ Facebook Ads                 ]    │
└──────────────────────────────────────┘
```

- **Название**: `My Campaign`
- **Алиас**: автогенерируется (можно перегенерировать 🎲)
- **Домен**: выберите созданный домен
- **Источник трафика**: (опционально) выберите источник

### Код для лендингов

Скопируйте код и разместите на лендинге:

```html
<span id="j4bCV6mX-xyz123"></span>
<script type="application/javascript">
document.getElementById('j4bCV6mX-xyz123').innerHTML = '<a href="https://tracker.example.com/j4bCV6mX?&se_referrer=' + encodeURIComponent(document.referrer) + '&default_keyword=' + encodeURIComponent(document.title) + '&'+window.location.search.replace('?', '&')+'">Link</a>';
</script>
```

### Вкладка "Финансы"

Выберите модель вознаграждения:
- **CPC** — оплата за клик
- **CPuC** — оплата за уникальный клик
- **CPM** — оплата за тысячу показов
- **CPA** — оплата за действие
- **CPS** — оплата за продажу
- **RevShare** — процент от дохода

Настройте уникальность:
- **Метод**: IP / IP+User-Agent / Cookies
- **Срок**: 24 часа

### Вкладка "Параметры"

Настройте до 37 параметров:
- `keyword` — ключевик
- `cost` — расход
- `currency` — валюта
- `external_id` — внешний ID
- `creative_id` — ID креатива
- `ad_campaign_id` — ID рекламной кампании
- `source` — сайт
- `sub_id_1` .. `sub_id_30` — дополнительные параметры

### Вкладка "Интеграции"

Выберите интеграцию для быстрой настройки:
- 📘 Facebook
- 🎵 TikTok
- 🔍 Яндекс
- 🔎 Google Ads
- 💬 VK Ads

### Вкладка "S2S Postbacks"

Настройте отправку постбеков во внешние системы:
1. Нажмите **+ Добавить Postback**
2. Введите URL: `https://external-system.com/postback?clickid={subid}&sum={payout}`
3. Выберите метод: GET или POST
4. Укажите статусы: `lead,sale`

### Вкладка "Заметки"

Поле для личных заметок по кампании.

## 7. Настройка потоков

Потоки определяют, куда направляется трафик.

### Типы потоков

| Тип | Описание | Приоритет |
|-----|----------|-----------|
| Перехватывающий | Для особых случаев | 1 (высший) |
| Обычный | Основной трафик | 2 |
| Замыкающий | Fallback | 3 (низший) |

### Создание потока

1. В правой части редактора нажмите **+ Обычный**
2. Настройте поток:
   - **Имя**: `Main Stream`
   - **Вес**: `100` (для ротации по весу)
   - **Вкл**: галочка

### Схема потока

Выберите тип схемы:

#### А) Прямой переход на оффер
```
[Campaign] → [Offer URL]
```
Выберите ID оффера в поле "ID Оффера".

#### Б) Лендинг + Оффер
```
[Campaign] → [Landing] → [Offer]
```
1. Нажмите **+ Добавить Лендинг**
2. Выберите лендинг из списка
3. Укажите вес (для сплит-теста)
4. Нажмите **+ Добавить Оффер**
5. Выберите оффер из списка

#### В) Действие
- **Do nothing** — пропустить клик
- **Показать 404** — вернуть 404
- **Показать HTML** — показать HTML код

### Фильтры потока

Добавьте фильтры для таргетинга:

1. Нажмите **+ Добавить фильтр**
2. Выберите тип фильтра:
   - **Country** — страна (RU, US, UK...)
   - **Device** — устройство (mobile, desktop, tablet)
   - **OS** — ОС (windows, macos, ios, android)
   - **Browser** — браузер (chrome, firefox, safari)
   - **Language** — язык (ru, en, de)
   - **ISP** — провайдер
   - **IP** — IP адрес
   - **Keyword** — ключевое слово
   - **Referer** — реферер
   - **Weekday** — день недели
   - **Time** — время

3. Выберите режим:
   - **Разрешить** — пропускать только указанные
   - **Запретить** — блокировать указанные

4. Введите значения через запятую: `RU, US, UK`

## 8. Получение ссылки

### Сохранение кампании

1. В header нажмите **Сохранить**
2. Дождитесь зелёной галочки ✓
3. Скопируйте URL кампании кнопкой 📋

### Формат ссылки

```
https://tracker.example.com/{alias}
```

Пример:
```
https://tracker.example.com/j4bCV6mX
```

### С параметрами

```
https://tracker.example.com/j4bCV6mX?keyword=casino&source=facebook&cost=0.50
```

## 9. Приём постбеков

### Настройка Postback

1. В меню выберите **Postback**
2. Скопируйте Postback URL:
   ```
   https://tracker.example.com/fd12e72/postback
   ```

### Параметры постбека

| Параметр | Описание | Обязательно |
|----------|----------|-------------|
| `subid` или `clickid` | ID клика | ✅ Да |
| `status` | Статус конверсии | ✅ Да |
| `payout` | Сумма выплаты | Нет |
| `currency` | Валюта | Нет |
| `tid` | Transaction ID | Нет |

### Поддерживаемые статусы

| Статус | Описание |
|--------|----------|
| `lead` | Лид |
| `sale` | Продажа |
| `rejected` | Отмена |
| `registration` | Регистрация |
| `deposit` | Депозит |
| `trash` | Нерелевантная |

### Пример постбека

```
GET https://tracker.example.com/fd12e72/postback?subid=abc123&status=sale&payout=50.00&currency=USD
```

### Тестирование

1. В разделе Postback нажмите **Тестировать**
2. Введите SubID существующего клика
3. Нажмите **Отправить тест**
4. Проверьте результат в логе конверсий

## 10. Просмотр статистики

### Dashboard

На главной странице отображаются:
- 7 карточек метрик
- График кликов/конверсий по дням
- 4 таблицы: Кампании, Офферы, Лендинги, Источники

### Карточки метрик

| Метрика | Описание |
|---------|----------|
| Клики | Общее количество кликов |
| Уник. клики | Уникальные по IP |
| Конверсии | Количество конверсий |
| Доход | Сумма выплат |
| Расход | Затраты на трафик |
| Прибыль | Доход - Расход |
| ROI | (Прибыль / Расход) × 100% |

### Лог кликов

1. В меню выберите **Логи** (или на Dashboard внизу)
2. Просмотрите последние клики

### Лог конверсий

1. В меню выберите **Конверсии**
2. Используйте фильтры:
   - По статусу
   - По кампании
   - По дате
   - Поиск по SubID

## 12. Администрирование

### Пользователи

1. В меню выберите **Администрирование → Пользователи**
2. Действия:
   - Создать пользователя
   - Редактировать права
   - Генерировать API ключи
   - Деактивировать

### Роли

| Роль | Права |
|------|-------|
| admin | Полный доступ |
| manager | Кампании, офферы, статистика |
| user | Только просмотр |

### Брендинг

1. Выберите **Администрирование → Брендинг**
2. Настройте:
   - Тёмная/Светлая тема
   - Цветовая схема (6 вариантов)
   - Кастомные цвета

### Гео-профили

1. Выберите **Администрирование → Гео-профили**
2. Используйте шаблоны:
   - USA and Canada
   - West Europe
   - Europe
   - exUSSR
   - English-Speaking
   - и другие...

### Состояние системы

1. Выберите **Администрирование → Состояние**
2. Просмотрите:
   - Версию трекера
   - Размер БД
   - Количество кликов/конверсий
   - Загрузку CPU

### Миграции

1. Выберите **Администрирование → Миграции**
2. Просмотрите статус миграций
3. Запустите ожидающие миграции

### Архив

1. Выберите **Администрирование → Архив**
2. Просмотрите архивированные элементы
3. Восстановите или удалите навсегда

## ❓ Частые вопросы

### Q: Как сменить пароль admin?
A: Администрирование → Пользователи → Редактировать admin → Ввести новый пароль

### Q: Как добавить свой источник трафика?
A: Источники трафика → Создать → Выбрать "Свой источник"

### Q: Почему не проходят постбеки?
A: Проверьте:
1. Правильность Postback URL
2. Существование клика с указанным subid
3. Логи постбеков в Администрировании

### Q: Как очистить статистику?
A: Откройте кампанию → Меню ⋮ → Очистка статистики

### Q: Как сделать сплит-тест офферов?
A: В потоке выберите "Лендинг + Оффер" → Добавьте несколько офферов с разными весами
